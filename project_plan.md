# Portfolio Website Project Plan

## Goal
Build a clean, professional personal portfolio website for job applications and networking.

## Main purpose
- Showcase my background
- Present my skills clearly
- Provide a live website link for recruiters
- Add projects gradually over time (currently using placeholder)

## Target audience
- Recruiters
- Hiring managers
- Networking contacts
- Potential employers in software, web, IT, and data-related roles

## Tech stack
- Next.js
- TypeScript
- Tailwind CSS
- GitHub
- Netlify

## MVP scope
Initial version should include:
- Home page
- About page
- Projects page with placeholders
- Contact page
- Resume download button
- Responsive layout

## Design style
- Clean
- Modern
- Minimal
- Professional
- Recruiter-friendly
- No excessive animation

## Current status
- Initial website scaffold created
- Local development working

## Next steps
- Refine homepage content
- Improve About page
- Add placeholder Projects section
- Polish Contact page
- Deploy to Netlify

## Future enhancements
- Add real projects
- Add dark mode
- Add simple contact form
- Improve visual polish
